# 📄 프로젝트 소개
 react와 firebase실습을 위해 진행한 클론코딩 사이트입니다

<br>

# ⌛ 제작 기간
2022년 7월 24일 ~ 2022년 8월 1일
<br>

# ⚙ 사용 기술 및 라이브러리
- `HTML / CSS`
- `Javascript`
- `React`
<br>

# 🛠 주요 기능
1. 회원가입
- 이메일과 비밀번호로 회원가입이 가능하며, 구글 및 깃허브로 소셜로그인 또한 가능합니다
2. 글쓰기
- 회원은 글을 쓸 수 있고 사진을 첨부할 수 있습니다. 또한 작성한 게시글은 삭제 및 수정이 가능합니다
3. 회원정보 수정
- 회원은 자신의 닉네임을 변경할 수 있습니다
